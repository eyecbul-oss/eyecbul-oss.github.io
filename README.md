# SezR Focus Premium Native V2

Bu paket eski web görünümlü yapıyı modern React Native / Expo uygulama mimarisine taşır.

## İçerik
- Gerçek bottom tab navigation
- Koç / Görev / Sayaç / Sınav / Ritim ekranları
- Premium koyu lacivert + sarı vurgu
- Glass card sistemi
- Neon buton sistemi
- Zustand store
- Focus timer hook
- Modern görev kartları

## Kurulum
npm install
npm start

## Android
npm run android
