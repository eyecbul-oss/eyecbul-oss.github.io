SEZR MATEMATİK - ESKİ ÇALIŞAN TAM SÜRÜM

Yeni GitHub repo için:
1) Repo içini boş bırak veya eski dosyaları sil.
2) Bu ZIP içindeki TÜM dosyaları repo ana dizinine yükle.
3) Commit changes.
4) Settings > Pages > Branch: main / root seç.
5) Site açıldıktan sonra tarayıcı önbelleğini temizle.

Ana dosyalar:
- index.html
- dersler.html
- youtube.html
- pdf.html
- focus.html
- iletisim.html
- style.css
- app.js
- manifest.json
- sw.js
- offline.html

Bu paket, son bozulan denemelerden önceki çalışan temiz sürüm temel alınarak hazırlandı.
